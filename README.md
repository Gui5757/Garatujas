# Garatujas

Serviços do Bun (https://bun.com/)

* Instalação do Bun (Linux):
    curl -fsSL https://bun.sh/install | bash

* Instalação do Bun (Windows):
    powershell -c "irm bun.sh/install.ps1 | iex"

* Verificar versão do Bun:
    bun -v

* Rodar o Bun e instalação de pacotes:
    bun init -y

Notas

async → transforma a função em assíncrona (retorna Promise)
await → pausa a execução até a Promise terminar
